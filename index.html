    About Binary Search Trees

    What is a BST?

    A Binary Search Tree is a node-based data structure where each node has at most two children. Every node follows a strict ordering rule based on its value.

    BST Property

    For any given node N:
    All values in the left subtree < N
    All values in the right subtree > N
    This property holds for every node in the tree.

    Real-World Uses

    BSTs are used in database indexing, autocomplete systems, file system directory trees, and anywhere you need fast sorted-order lookups.

    Time Complexity
